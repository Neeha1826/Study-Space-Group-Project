# Study-Space-Group-Project

This is a git repo for a project called StudySense, made by Cardiff University students for our Group Project and Internet of Things modules.

This project is for an IoT device which is used in libraries and study spaces to calculate the  occupancy numbers and environmental metrics, which eventually reach the user via the included website.  This allows users to find study spaces that are suitable for them, based off of the information colllected by the device.

This project uses a raspberry pi 4b device on bullseye 64, a grovepi board using the DHT and Noise sensors and a pi-camera
